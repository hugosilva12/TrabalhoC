◆ lerData()

void lerData ( Data *  data,
int  AnoMax 
)

Lê da consola uma data

Parâmetros
dataapontador para uma estrutura data que contém três valores inteiros, um é referido ao dia, outro ao mês e outro ao ano. Esta função verifica os dados inseridos pelo utilizador de modo a que os correspondam à realidade.Por exemplo é impossível ao utilizador registar uma data de entrada com o ano 2030, sendo que o ano atual é 2021.
